
# Create: Radiation & Fission

This mod, akin to how it sounds, promises to add all kinds of nuclear-based content into Create. For now, our scope is limited to fission-based power generation, which on its own is fun to play around with, and could add a lot more depth in the future of other create addons. 

## Mod Biography:
Create Radiation and Fission has been in a constant on-off cycle of concepting and development. This is because the dev(s) all have a pretty full life, and when pressure escalates, progress can slow, and the project has sufferered because of it. But don't give up hope, as I, Eevneon, have regained motivation in the last few years, and am seriously having fun developing this mod.

## A short list of planned features:
- A fun uranium enrichment process that scales with effort,
- At least one semi-realistic fission reactor based on enriched uranium,
- Redstone components to fit the difficult controlling of a fission reactor
- Huge amounts of su,
- And much, much more, soon to be announced!
